Selects


Templar will use each element in the list as both the name and value


You can specify different names and values by binding to a list of objects that have

name and value properties


Once a list is bound to a select, it gets a "current_selection" property that can be used to

get/set the value of the currently selected option.


Simple select's current selection is {{Example.SimpleSelect.current_selection}}

Complex select's current selection is {{Example.SimpleSelect.current_selection}}


To set a select's current selection, assign 'current_selection' to a value that is in the list.

Remember that for simple lists, text and value are the same, but for complex lists the value may

be different than the text and the set will only work for valid values.
